Node.js est un environnement d'exécution JavaScript côté serveur, qui permet d'exécuter du code JavaScript en dehors du navigateur web. Node.js utilise le moteur JavaScript V8 de Google pour exécuter du code JavaScript, et permet de développer des applications réseau évolutives et en temps réel. Node.js utilise un modèle d'entrée-sortie asynchrone non bloquant qui permet une utilisation efficace des ressources système.
